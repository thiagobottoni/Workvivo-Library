Popular Questions
How do I get IT help?
For all non-urgent issues, the best way to get help is to Open an IT Ticket using our self-service portal. This ensures your request is tracked and assigned to the right team. For urgent issues that impact business operations, please call the IT hotline.
What is the IT hotline number?
Our 24/7 IT hotline is available for critical system outages or security incidents. You can reach us at +55 (11) 5555-1111. Please have your employee ID ready when you call.
How do I reset my password?
You can reset your network password at any time using our Password Reset Tool. You will be asked to verify your identity with security questions you've previously set up. If you are unable to use the tool, please submit a ticket.
How can I request new software?
All software requests must be submitted through the Service Catalog. Browse for the application you need and complete the request form. Please note that some software requires manager approval, which is handled automatically through the system.
